# How to Use
## How to set the time
### By write the time down 
- Please write down the time that you want to start

### By the buttons
- If you push the buttons, you can add the time.

#### +10 button
- you can add 10 minutes.

#### +5 button
- you can add 5 minutes.

## Start Button
- start the timer with the set time.

## Stop button
- stop the timer.

## Reset button
- reset the timer.
